# 🔐 Cryptography Project — Caesar & Vigenère Ciphers (Python)

This project implements classical encryption algorithms using Python.

The goal of the project is to understand how historical cipher systems work and how they can be implemented, decoded and even brute-forced using programming.

It includes implementations of:

- Caesar Cipher encoding and decoding  
- Brute force Caesar Cipher cracking  
- Vigenère Cipher encoding and decoding  
- Message processing with spaces and punctuation preserved  

---
### Caesar Cipher

A monoalphabetic substitution cipher where each letter in the message is shifted by a fixed offset in the alphabet.

Functions included:

- `vigenere_encode(message, keyword)`
- `vigenere_decode(message, keyword)`
